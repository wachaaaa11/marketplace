# Архитектура веб-приложения объявлений

## Технологический стек

### Фронтенд
- **HTML5** - современная семантическая разметка
- **CSS3** - стили с использованием Grid и Flexbox
- **JavaScript ES6+** - современный синтаксис, модули, async/await
- **Tailwind CSS** - utility-first CSS framework для быстрой стилизации

### Бэкенд
- **Node.js** - JavaScript runtime для серверной разработки
- **Express.js** - минималистичный веб-фреймворк для Node.js
- **JWT** - аутентификация через JSON Web Tokens
- **bcrypt** - хеширование паролей

### База данных
- **SQLite** - встроенная реляционная СУБД (обоснование выбора ниже)

## Обоснование выбора SQLite

1. **Реляционная структура**: Полная поддержка SQL со связями между таблицами (users → ads → images, users → messages)
2. **Целостность данных**: Включение PRAGMA foreign_keys = ON обеспечивает ссылочную целостность
3. **Простота разработки**: Весь проект (database.db) — один файл, не требует отдельного сервера БД
4. **Автономность**: Идеально для прототипов и проектов с умеренной нагрузкой
5. **Надежность**: ACID-совместимость гарантирует сохранность данных объявлений и сообщений

## Основные функции

1. **Аутентификация и авторизация**
   - Регистрация новых пользователей
   - Вход в систему
   - JWT токены для безопасности

2. **Управление объявлениями**
   - Создание новых объявлений
   - Редактирование своих объявлений
   - Удаление объявлений
   - Публикация/снятие с публикации

3. **Поиск и фильтрация**
   - Поиск по ключевым словам
   - Фильтрация по категориям и подкатегориям
   - Фильтрация по цене, местоположению
   - Сортировка по дате и цене

4. **Медиа контент**
   - Загрузка нескольких изображений
   - Оптимизация изображений
   - Галерея на детальной странице


## Безопасность

- Ограничение размера загружаемых файлов
- Аутентификация для защищенных маршрутов
