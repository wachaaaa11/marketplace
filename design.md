# Дизайн веб-приложения объявлений

## Дизайн-философия

### Цветовая палитра
- **Основной цвет**: #2563eb (синий) - для кнопок и акцентов
- **Вторичный цвет**: #64748b (серый) - для второстепенных элементов
- **Фон**: #f8fafc (светло-серый) - основной фон страниц
- **Текст**: #1e293b (темно-серый) - основной цвет текста
- **Успех**: #10b981 (зеленый) - для успешных действий
- **Ошибка**: #ef4444 (красный) - для ошибок и предупреждений

### Типографика
- **Заголовки**: Inter, sans-serif (современный, читаемый)
- **Основной текст**: Inter, sans-serif
- **Размеры**: 
  - H1: 2.5rem (40px)
  - H2: 2rem (32px)
  - H3: 1.5rem (24px)
  - Основной текст: 1rem (16px)
  - Мелкий текст: 0.875rem (14px)

## Компоненты интерфейса

### 1. Главная страница (index.html)
**Макет**: 
- Шапка с логотипом, поиском и навигацией
- Героический блок с призывом к действию
- Быстрые категории (иконки с названиями)
- Популярные объявления (карточки товаров)
- Призыв к регистрации

**Интерактивные элементы**:
- Поиск с автоподсказками
- Фильтры по категориям
- Карусель популярных объявлений
- Анимированные иконки категорий

### 2. Страница объявлений (ads.html)
**Макет**:
- Боковая панель фильтров (25%)
- Основная область с сеткой карточек (75%)
- Пагинация внизу

**Фильтры**:
- Категории (древовидный список)
- Ценовой диапазон (ползунки)
- Местоположение
- Состояние товара (новое/б/у)
- Сортировка (по дате, цене)

### 3. Детальная страница объявления (ad-detail.html)
**Макет**:
- Галерея изображений (60%)
- Информация о товаре (40%)
- Описание
- Контактная информация продавца
- Кнопка "Написать сообщение"

**Интерактивные элементы**:
- Лайтбокс для изображений
- Карта местоположения
- Форма отправки сообщения

### 4. Создание объявления (create-ad.html)
**Макет**:
- Многоступенчатая форма
- Загрузка изображений (drag & drop)
- Предпросмотр объявления

## Визуальные эффекты

### Используемые библиотеки
1. **Anime.js** - анимации элементов при загрузке и взаимодействии
2. **Splide.js** - карусели изображений
3. **ECharts.js** - визуализация статистики (если потребуется)
4. **p5.js** - фоновые анимации на главной странице

### Эффекты
- **Плавная прокрутка**: scroll-behavior: smooth
- **Анимация появления**: fade-in для карточек при скролле
- **Hover эффекты**: тень и подъем карточек при наведении
- **Загрузка изображений**: skeleton screens
- **Микро-анимации**: для кнопок и интерактивных элементов

### Фоновые эффекты
- **Главная страница**: анимированные геометрические фигуры с помощью p5.js
- **Страницы объявлений**: чистый фон с акцентами
- **Детальные страницы**: фокус на контенте, минимальные эффекты

## Адаптивный дизайн
- **Desktop**: 1200px+
- **Tablet**: 768px - 1199px

- **Модальные окна**: для авторизации и просмотра изображений
- **Toast уведомления**: для обратной связи
